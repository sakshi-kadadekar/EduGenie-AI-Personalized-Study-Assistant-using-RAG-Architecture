# EduGenie AI - Architecture

## Overview

EduGenie AI is a full-stack educational platform powered by artificial intelligence.

## Components

### Frontend
- **Framework**: Next.js with TypeScript
- **Styling**: Tailwind CSS
- **Location**: `/frontend`

### Backend
- **Framework**: FastAPI
- **Language**: Python
- **Location**: `/backend`

### AI Engine
- **Purpose**: ML models and NLP processing
- **Location**: `/ai-engine`

## Project Structure

```
edugenie-ai/
├── frontend/        # Next.js frontend application
├── backend/         # FastAPI backend services
├── ai-engine/       # AI/ML models and processing
├── docs/            # Documentation
└── deployment/      # Deployment configurations
```

- `/auth/*` - Authentication routes
